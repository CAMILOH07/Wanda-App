Presentación Aplicacion Movil Wanda

Introducción
Este software fue desarrollado con el objetivo de mejorar la comunicación académica y la colaboración entre los estudiantes de la universidad. A diferencia de las redes sociales tradicionales, la aplicación se centra exclusivamente en la vida universitaria y aprovecha el correo institucional para garantizar autenticidad, seguridad y un entorno más controlado.

¿Qué es la aplicación?
Es una plataforma digital que permite:

Registrarse con correo institucional o cualquier correo válido.

Agregar amigos dentro de la comunidad universitaria.

Subir, compartir y descargar archivos académicos.

Chatear de manera instantánea con otros estudiantes.

Mantener un ecosistema académico cerrado, confiable y más seguro que redes sociales externas.
